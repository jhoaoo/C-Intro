# C++ Intro

Repositorio introductorio de programación en **C++**.

## Descripción

Este repositorio contiene ejercicios iniciales orientados a reforzar fundamentos de programación, sintaxis, estructuras de control, funciones y resolución de problemas básicos mediante C++.

## Objetivos

- Practicar sintaxis básica de C++.
- Resolver ejercicios introductorios.
- Fortalecer lógica de programación.
- Usar GitHub como respaldo del aprendizaje.

## Temas trabajados

- Entrada y salida de datos.
- Variables y tipos de datos.
- Condicionales.
- Bucles.
- Funciones.
- Organización básica de programas.
